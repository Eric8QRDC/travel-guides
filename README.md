# 青森紅葉 9 日旅遊指南 · 2026 秋

> **Aomori Autumn Foliage 9-Day Travel Guide · 2026**

花卷進函館出的青森紅葉深度旅遊指南，虎航全程。涵蓋奧入瀾東段、城ヶ館温泉、HOTEL Jogakura、弘前城 + Montbell 購物、函館觀光（五稜郭、元町、函館山夜景）。

## 📖 線上閱讀

👉 **https://Eric8QRDC.github.io/travel-guides/**

## ✨ 內容

- **9 日完整行程**（D1–D9）含時刻表
- **兩點深度策略**：青森 4 晚 + 函館 2 晚為主基地，中間穿插十和田湖 1 晚 + Jogakura 1 晚
- **花卷進函館出虎航全程**（IT258 + IT237）
- **城ヶ倉大橋 + 八甲田纜車**紅葉絕景
- **弘前城 + Montbell 弘前店**購物一日遊
- **函館觀光整天**：五稜郭タワー + 元町 + 函館山夜景巴士
- **HOTEL Jogakura 免費接送**（八甲田纜車 / 城ヶ倉大橋 / 酸ヶ湯）
- **預算 ¥190,000**（約 NT$40,000，單人估）

## 🗓 行程速覽

```
D1 (10/31 六)  桃園 → 花卷 → 盛岡 → 新青森 → 青森
D2 (11/1 日)   青森 → 燒山 → 銚子大滝 → 子ノ口 → 十和田湖泊
D3 (11/2 一)   十和田湖 → 巴士 → HOTEL Jogakura 城ヶ館
D4 (11/3 二)   城ヶ倉大橋日出 → 八甲田纜車 → 青森
D5 (11/4 三)   青森 → JR → 弘前（弘前城 + Montbell）→ 青森
D6 (11/5 四)   青森市區悠閒（美術館 + 味噌咖哩 + A-FACTORY）
D7 (11/6 五)   青森 → はやぶさ → 函館
D8 (11/7 六)   函館觀光（五稜郭 + 元町 + 函館山夜景）
D9 (11/8 日)   函館朝市 → IT237 → 桃園
```

## ✈️ 航班須知

桃園 ⇄ 花卷由台灣虎航 IT258 營運（10/31 冬季班請上官網查證），函館 ⇄ 桃園由 IT237 營運（週一三四五日，11/8 為週日需自查時刻是否調整）。

## ⚠️ 關鍵風險

- **IT258 桃園 → 花卷 10/31（六）**冬季班待查證
- **函館山纜車 2026/10/20–11/8 停運**，D8 夜景改搭夜景ダイレクトバス
- **八甲田纜車整備運休 11/8–11/13**，D4 仍夏季班末班 16:20
- **十和田湖遊覽船 11/16 收班**，D2/D3 為最後機會
